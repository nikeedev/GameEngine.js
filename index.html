Meet GameEngine.js

GameEngine.js, is a new, and modern 2D game engine, that runs using Canvas API element.
It is light, easy, and has functions that are useful in your future game

Here are examples that are using the engine( source code could be found on GitHub):

Example 1: This example are showing how you can combine all different stuff from engine, to make a rainbow text and box moving on canvas. WARNING, FLASHING CONTENT!

Example 2: This example are showing how you can combine all different stuff from engine, to make a COOL box moving down and up, that also changes color and size.

Example 3: This is an another example that are showing how you can combine all different stuff from engine, to make a drawing website. Draw by holding left mouse button down, and moving mouse around, same like MS Paint. Click Space to clear canvas.

Example 4: Example result, where you can see how engine, draws a sprite and updates it position.

Example 5: Example of using game.setScreenMode("max");, which sets screen size automatically to big, but set sets a bit smaller to configure to the window size.

Example 6: Example result of getting key inputs by the engine, that way you can control the square in the canvas.

Example 7: Example result of playing sound using the engine. Play sound by clicking the screen.
